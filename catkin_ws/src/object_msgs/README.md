# object_msgs
## Introduction
This package defines the object messages used by object detection.

```
# Building
cd ~/catkin_ws/src
git clone https://github.com/intel/object_msgs
cd ~/catkin_ws
catkin_make
# Installation
catkin_make install
source install/setup.bash
```
